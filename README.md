<h1>Escalator(Stack) Project</h1>
<p>
  Create a Stack class that uses an array to hold elements. The class will include the following methods:<br>
  1. push(): Adds an element to the top of the stack.<br>
  2. pop(): Removes the top element from the stack.<br>
  3. top(): Returns the top element without removing it.<br>
  4. isEmpty(): Checks if the stack is empty.<br>
  5. isFull(): Checks if the stack is full.
</p>
